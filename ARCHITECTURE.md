### Network Details

| Component | Config |
|-----------|--------|
| VPC CIDR | `10.0.0.0/16` |
| Availability Zones | 2 (auto-selected) |
| Public subnets | `10.0.0.0/24`, `10.0.1.0/24` вҖ” NAT Gateway + Internet Gateway |
| Private subnets | `10.0.10.0/24`, `10.0.11.0/24` вҖ” ALB, ECS tasks, VPC Link |
| NAT Gateway | 1x in public subnet 0 (cost-optimized for dev) |
| ALB | Internal, HTTP :80, private subnets only |

### Security Groups

| Security Group | Inbound | Outbound |
|----------------|---------|----------|
| `alb-sg` | HTTP :80 from VPC CIDR | All |
| `ecs-tasks-sg` | TCP :8081-8084 from ALB SG only | All (AWS API calls via NAT) |
| `vpclink-sg` | вҖ” | TCP :80 to ALB SG |

### API Gateway Routes

| Route | Auth | Target |
|-------|------|--------|
| `POST /api/users/register` | None | ALB вҶ’ user-service |
| `POST /api/users/confirm` | None | ALB вҶ’ user-service |
| `POST /api/users/login` | None | ALB вҶ’ user-service |
| `POST /api/users/refresh` | None | ALB вҶ’ user-service |
| `GET /api/plans` | None | ALB вҶ’ plan-catalog-service |
| `GET /api/plans/{planId}` | None | ALB вҶ’ plan-catalog-service |
| `POST /api/agent/chat` | JWT | Lambda (chat_api) |
| `ANY /api/{proxy+}` | JWT | ALB (catch-all) |

### ALB Path-Based Routing

| Path Pattern | Target |
|-------------|--------|
| `/api/users*` | user-service :8081 |
| `/api/plans*` | plan-catalog-service :8082 |
| `/api/orders*` | order-service :8083 |
| Default | 404 JSON response |
